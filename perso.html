Projet Personnel

  • Gestionnaire de mot de passe

    Voici le menue d'un gestionnaire de mot de passe avec une IHM tactile que j'ai crée en python avec la librairie pygame que j'ai appelée Shirka. Le menu principal est composé de 9 sous-menus :

    • Un menus "aide" résumant les fonctions des menus
    • Un menus "liste" listant les identifiant et si cliqué met dans le presse papier le mot de passe associer
    • Un menus "nouveau" pour créer une nouvelle paire identifiant-mot de passe
    • Un menus "changer" pour changer un identifiant ou un mot de passe
    • Un menus "mot de passe" pour obtenir le mot de passe d'un identifiant
    • Un menus "créer" qui génère un mot de passe aléatoire pour un identifiant de compte
    • Un menus "supprimer" pour supprimer une paire identifiant-mot de passe
    • Un menus "téléverser" pour télécharger les mots de passes dans un fichier txt
    • Et enfin un menus "paramètre" pour modifier la question à poser en cas de perte de mot de passe, le mot de passe de Shirka ou réinstialiser les données.

  • Loups-garous

    J'ai développé en Java avec de la programmation orienté objet, un programme qui simule des parties de Loups-Garous. Après avoir sélectionner les options d'affichage le nombre de simples villageois, de simples loups-garous et les villageois spécial que l'on veut ajouter, on peut lançer la partie

    Ces parties me permettent d'analyser l'impact des personnages et ainsi estimer leurs forces afin d'améliorer l'équilibre d'une partie de Loups-Garous
